# My Mortgage Calculator (React - JavaScript)

This is a simplified mortgage calculator wireframe built with React (JavaScript).

## How to run locally

1. Ensure you have Node.js (>=16) and npm or yarn installed.
2. In the project root directory, run:
   ```
   npm install
   npm start
   ```
   or with yarn:
   ```
   yarn
   yarn start
   ```
3. Open http://localhost:3000 in your browser.

## Notes
- This project is intentionally minimal and uses plain JavaScript (no TypeScript).
- The UI is focused on accessibility and responsiveness.
